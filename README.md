# Project-Tic-Tac-Toe-Game
🎮 Tic-Tac-Toe Game A clean, interactive, and responsive Tic-Tac-Toe game built using web technologies. This project focuses on DOM manipulation, game logic implementation, and responsive UI design.
✨ Features
Interactive UI: Smooth animations and hover effects for a better user experience.

Game Logic: Robust logic to detect win patterns (rows, columns, diagonals) and draw scenarios.

Responsive Design: Fully functional on desktops, tablets, and mobile devices.

Reset Functionality: Quickly restart the game with a single click.

Turn Indicator: Real-time display to show which player's turn it is (X or O).

🛠️ Tech Stack
HTML5: For the structural layout of the game board.

CSS3: For styling, Flexbox layouts, and transitions.

JavaScript (ES6): For handling game state, click events, and winning algorithms.
